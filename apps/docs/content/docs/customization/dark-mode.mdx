## Dark Mode Overview

Principium UI provides comprehensive dark mode support through its theme system. Dark mode is automatically handled by the `shadesPlugin` and integrates seamlessly with popular theme providers like `next-themes`.

## Quick Setup

### Basic Dark Mode Configuration

<SyntaxHighlighter className="bg-border" language="typescript" code={`// tailwind.config.ts
import {shadesPlugin} from '@principium/theme';

const config = {
plugins: [
shadesPlugin({
defaultTheme: 'light',
themes: {
light: {
colors: {
background: { hue: 0, saturation: 0 },
primary: { hue: 212, saturation: 100 },
// ... other light theme colors
},
},
dark: {
isDarkTheme: true, // ← Important: marks this as a dark theme
colors: {
background: { hue: 220, saturation: 15 },
primary: { hue: 212, saturation: 85 },
// ... other dark theme colors
},
},
},
}),
],
};`} />

### Theme Provider Setup

<SyntaxHighlighter className="bg-border" language="tsx" code={`// app/providers.tsx
'use client';

import {ThemeProvider} from 'next-themes';

export function Providers({children}: {children: React.ReactNode}) {
  return (
    <ThemeProvider
      attribute="class"
      defaultTheme="system"
      themes={['light', 'dark']}
      enableSystem={true}
    >
      {children}
    </ThemeProvider>
  );
}`} />

        </Providers>
      </body>
    </html>
  );
}`} />

## Dark Mode Colors

### Color Adaptations

Dark themes typically need adjusted saturation and different color relationships:

<SyntaxHighlighter
  className="bg-border"
  language="typescript"
  code={`const colorComparison = {
  light: {
    // Light theme - higher saturation, lighter backgrounds
    background: { hue: 0, saturation: 0 },    // Pure gray
    primary: { hue: 212, saturation: 100 },   // Vivid blue
    success: { hue: 146, saturation: 55 },    // Standard green
    border: { hue: 240, saturation: 10 },     // Cool gray
  },
  dark: {
    // Dark theme - lower saturation, darker backgrounds
    background: { hue: 220, saturation: 15 }, // Dark blue-gray
    primary: { hue: 212, saturation: 85 },    // Less saturated blue
    success: { hue: 146, saturation: 45 },    // Muted green
    border: { hue: 220, saturation: 20 },     // Blue-tinted borders
  },
};`}
/>

### Dark Mode Best Practices

#### 1. Reduce Saturation

Pure colors can be harsh in dark mode:

<SyntaxHighlighter className="bg-border" language="typescript" code={`// ✅ Good - reduced saturation for dark mode
dark: {
  colors: {
    primary: { hue: 212, saturation: 85 },    // Was 100 in light
    success: { hue: 146, saturation: 45 },    // Was 55 in light
    warning: { hue: 37, saturation: 75 },     // Was 90 in light
  },
}

// ❌ Avoid - same saturation as light mode
dark: {
colors: {
primary: { hue: 212, saturation: 100 }, // Too harsh
success: { hue: 146, saturation: 55 }, // Too bright
},
}`} />

#### 2. Tinted Backgrounds

Pure black backgrounds can be harsh. Use dark grays with subtle color tints:

<SyntaxHighlighter className="bg-border" language="typescript" code={`// ✅ Good - tinted dark backgrounds
background: { hue: 220, saturation: 15 }, // Dark blue-gray
border: { hue: 220, saturation: 20 },     // Matching tint

// ❌ Avoid - pure black/gray
background: { hue: 0, saturation: 0 }, // Too stark`} />

#### 3. Maintain Contrast

Ensure sufficient contrast ratios in dark mode:

<SyntaxHighlighter
  className="bg-border"
  language="typescript"
  code={`// Light mode text uses background-900 (very dark)
// Dark mode text uses background-100 (very light)
// This happens automatically with the color scale`}
/>

## Theme Toggle Component

### Basic Toggle

<SyntaxHighlighter className="bg-border" language="tsx" code={`'use client';

import {useTheme} from 'next-themes';
import {Button} from '@principium/react';
import {SunIcon, MoonIcon} from '@phosphor-icons/react';

export function ThemeToggle() {
  const {theme, setTheme} = useTheme();

return (
<Button
variant="ghost"
size="icon"
onClick={() => setTheme(theme === 'dark' ? 'light' : 'dark')} >
<SunIcon className="h-5 w-5 rotate-0 scale-100 transition-all dark:-rotate-90 dark:scale-0" />
<MoonIcon className="absolute h-5 w-5 rotate-90 scale-0 transition-all dark:rotate-0 dark:scale-100" />
<span className="sr-only">Toggle theme</span>
</Button>
);
}`} />

### Advanced Toggle with System

<SyntaxHighlighter className="bg-border" language="tsx" code={`'use client';

import {useTheme} from 'next-themes';
import {Button} from '@principium/react';
import {SunIcon, MoonIcon, DesktopIcon} from '@phosphor-icons/react';

const themes = [
{ key: 'light', icon: SunIcon, label: 'Light' },
{ key: 'dark', icon: MoonIcon, label: 'Dark' },
{ key: 'system', icon: DesktopIcon, label: 'System' },
];

export function ThemeSelector() {
  const {theme, setTheme} = useTheme();

return (
<div className="flex rounded-lg border border-border-200 p-1">
{themes.map(({key, icon: Icon, label}) => (
<Button
key={key}
variant={theme === key ? 'primary' : 'ghost'}
size="sm"
onClick={() => setTheme(key)}
className="flex items-center gap-2" >
<Icon size={16} />
{label}
</Button>
))}
</div>
);
}`} />

## Troubleshooting

### Hydration Issues

Prevent hydration mismatches:

<SyntaxHighlighter className="bg-border" language="tsx" code={`'use client';

import {useTheme} from 'next-themes';
import {useEffect, useState} from 'react';

export function ThemeAwareComponent() {
  const {theme} = useTheme();
  const [mounted, setMounted] = useState(false);

useEffect(() => {
setMounted(true);
}, []);

// Return a skeleton or neutral version until mounted
if (!mounted) {
return (
<div className="h-8 w-8 rounded bg-background-200 animate-pulse" />
);
}

return (
<div className={\`themed-component theme-\${theme}\`}>
Theme-specific content
</div>
);
}`} />

### Flash of Unstyled Content (FOUC)

Prevent FOUC by setting the theme class on the html element:

<SyntaxHighlighter className="bg-border" language="tsx" code={`// app/layout.tsx
import Script from 'next/script';

export default function RootLayout({children}) {
  return (
    <html lang="en" suppressHydrationWarning>
      <head>
        <Script id="theme-script" strategy="beforeInteractive">
          {\`
            try {
              const theme = localStorage.getItem('theme') || 'system';
              const systemTheme = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
              const effectiveTheme = theme === 'system' ? systemTheme : theme;
              document.documentElement.classList.add(effectiveTheme);
            } catch (e) {}
          \`}
        </Script>
      </head>
      <body>
        <Providers>{children}</Providers>
      </body>
    </html>
  );
}`} />

### Screen Reader Announcements

<SyntaxHighlighter className="bg-border"  language="tsx" code={`'use client';

import {useTheme} from 'next-themes';
import {useEffect} from 'react';

export function ThemeAnnouncer() {
  const {theme} = useTheme();

useEffect(() => {
// Announce theme changes to screen readers
const announcement = \`Theme changed to \${theme}\`;
const announcer = document.createElement('div');
announcer.setAttribute('aria-live', 'polite');
announcer.setAttribute('aria-atomic', 'true');
announcer.className = 'sr-only';
announcer.textContent = announcement;

    document.body.appendChild(announcer);
