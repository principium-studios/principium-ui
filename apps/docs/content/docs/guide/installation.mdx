## 1. Install dependencies

Install the theme package (provides all utility classes used by components):

<PackageManagers packageName="theme" />

Then install the Shades plugin (hooks the theme into Tailwind and exposes config):

<PackageManagers packageName="shades" />

## 2. Configure Tailwind

Add this to your `tailwind.config.ts`:

<SyntaxHighlighter className="bg-border" language="ts" code={`import {shadesPlugin} from '@principium/shades';

export default {
    content: ['./node_modules/@principium/theme/dist/**/*.{js,ts,jsx,tsx}'],
    plugins: [
      shadesPlugin({
        defaults: true,
      }),
    ],
};
`} />

Note: adjust the `content` path if your project structure differs (e.g., monorepo).

## 3. Wire up global CSS

Add this to your `globals.css` (and ensure this stylesheet is loaded by your app):

<SyntaxHighlighter className="bg-border" language="css" code={`@import 'tailwindcss';

/* Make Principium Theme available to Tailwind */
@config "../tailwind.config.ts";
@custom-variant dark (&:is(.dark \*));
`} />

## 4. Add components

Install the React package to start importing or scaffolding components.  
You can also add components individually as needed-see the [components page](/docs/guide/components) for details.

<PackageManagers packageName="react" />
